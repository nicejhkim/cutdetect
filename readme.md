# detection prototype
(AI 모델 학습을 위해) 원본 영상에서 특정 클립을 잘라내는 작업의 프로토타입

# modules
기본적인 작업은
    1. 원본 영상을 scene 단위로 분리
    2. 각 scene 에서 대표 frame 추출
    3. 추출한 frame의 기반으로 scene의 구성(object) 분석
    4. 목적에 맞는 scene 영상으로 저장
    5. 영상 후처리 (비식별화 처리 등)
다섯 단계로 이루어진다.
